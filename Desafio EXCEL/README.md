# Bootcamp Santander - Dashboard de vendas Xbox Excel

Este projeto é um desafio de Dashboard de vendas de gamepass da Xbox utilizando Excel.

## Objetivo
Construir dashboard no Excel que explique claramente dados de vendas de passes diferentes oferecidos pelo serviço da Xbox Game Pass.

## Tecnologias Utilizadas
* **Excel**

## Estrutura do Projeto
* `Dashboard de Vendas Xbox.xlsx`: O dashboard já feito (os dados utilizados se encontra no mesmo arquivo)
